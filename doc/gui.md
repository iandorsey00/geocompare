# GUI

After completing [Setup](./setup.md), start the GUI:

```bash
geocompare-gui
```

## Main interface

The main window has two sections:

- top: search and profile lookup
- bottom: ranking and related comparison controls

If you do not know an exact display label, search first and pick from the results.

## Windows

The `DemographicProfile` window shows one geography in detail.

From that window you can open:

- highest and lowest rankings for individual identifiers
- physically nearest geographies
- demographically similar geographies

## Notes

The GUI is a convenience layer over the same core query functionality exposed by the CLI.
